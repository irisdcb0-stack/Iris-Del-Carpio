# ElectroChida — Página de reseñas y venta (Minimal)

Sitio estático minimalista con:

- `index.html` — página principal con productos, reseñas y carrito.
- `css/styles.css` — estilos modernos y responsivos.
- `js/script.js` — lógica mínima para carrito y envío de reseñas (simulado).
- `assets/` — imágenes SVG de ejemplo.

Para ver la página localmente, abre `index.html` en tu navegador. En PowerShell puedes ejecutar:

```powershell
ii .\index.html
```

Si quieres, puedo:
- Añadir más productos desde un JSON.
- Conectar un carrito real (backend / pago).
- Mejorar accesibilidad o SEO.
