# La Confitería Barroca — Proyecto Web (Responsive)

Proyecto de ejemplo: página web de pastelería y repostería con estilo barroco.

Estructura creada:

- index.html (home)
- about.html (nosotros)
- productos.html (productos / galería)
- recetas.html (recetas)
- contacto.html (formulario de contacto)
- css/estilos.css (estilos principales)
- js/main.js (interactividad: menú, lightbox, validación)
- images/ (SVGs de ejemplo: hero, cakes, pattern, logo)

Cómo abrir localmente:

1. Abrir la carpeta del proyecto en Visual Studio Code o tu editor preferido.
2. Abrir `index.html` en el navegador (doble clic o clic derecho -> "Abrir con...").

Notas y accesibilidad:

- HTML semántico con `header`, `nav`, `main`, `section`, `article`, `footer`.
- Atributos ARIA en menú y modal. Formularios con `aria-live` para feedback.
- Mobile-first; la navegación se adapta a pantalla pequeña con un botón hamburguesa.
